# 🎙️ Speech Assistant Pro

A modern **Text-to-Speech (TTS)** and **Speech-to-Text (STT)** desktop application built with **Python + Tkinter**.
Designed with a clean card-style UI and enhanced with powerful features for productivity and accessibility.

---

## ✨ Features

* 🔊 **Text to Speech**
  Convert typed text into natural-sounding speech.

* 🎤 **Speech to Text**
  Speak through your microphone and get real-time text output.

* 🎚️ **Voice Controls**

  * Adjust speaking speed
  * Control volume
  * Select different system voices

* 💾 **Save & Load Text**

  * Save text as `.txt` files
  * Load existing text files

* 🎧 **Export Audio**

  * Convert text into an audio file (`.mp3`)

* 🎙️ **Mic Status Indicator**

  * Visual feedback while listening

* 🧊 **Modern UI**

  * Dark theme
  * Card-style layout
  * Clean and minimal design

---

## 🛠️ Tech Stack

* **Python**
* **Tkinter** (GUI)
* **pyttsx3** (Text-to-Speech)
* **SpeechRecognition** (Speech-to-Text)
* **PyAudio** (Microphone input)

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/speech-assistant.git
cd speech-assistant
```

### 2. Install dependencies

```bash
pip install pyttsx3 SpeechRecognition pyaudio
```

👉 If `pyaudio` fails (Windows):

```bash
pip install pipwin
pipwin install pyaudio
```

---

## ▶️ Usage

Run the application:

```bash
python app.py
```

---

## 🎮 How to Use

1. Type text into the input box
2. Click **🔊 Speak** to hear it
3. Click **🎤 Listen** and speak to convert speech to text
4. Adjust **speed, volume, and voice** using sliders
5. Save or load text files as needed
6. Export speech as an audio file
